Hi! This is a CSS style i made to block the AI search buttons after finding out that Ecosia, for some reason, has no way to disable the "AI Search" buttons prevalent all over the search page. 

I initially tried using Ublock Origin, but couldn't find a way to make it cosmetic filter only (I like planting trees, just not asking AI how to plant trees if ya know what i mean)

Sooo here we are! The CSS is pretty bare-bones, it just selects the page elements using an ID attribute / class and tells them to make themself scarce. 
You'll probably see them flash when the page loads, but they should move on pretty much immediately

If i missed anything, feel free to make an issue! 
This was done in a few hours out of frustration so I make no promises to rigorously update it, but I'll keep updating it as I use it so hopefully that's enough :D
